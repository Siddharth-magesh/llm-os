====================================
  LLM-OS Quick Start Guide
====================================

LAUNCH OPTIONS:
====================================

Auto (Groq with Ollama fallback):
  launch.bat

Groq only (cloud, fast):
  launch_groq.bat

Ollama only (local, private):
  launch_ollama.bat

====================================
VERIFY INSTALLATION:
====================================

Run tests:
  tests\run_tests.bat

Quick check:
  verify.bat

====================================
PROVIDERS:
====================================

Groq (Cloud):
  - Model: llama-3.3-70b-versatile
  - Speed: ~100-200ms latency
  - Status: WORKING ✓

Ollama (Local):
  - Model: llama3.2:3b
  - Speed: Depends on hardware
  - Status: Available (if installed)
